# Python SDK (hebbs-python) — Feature Parity with Proto & CLI

This document tracks which proto/CLI features are exposed in the Python gRPC SDK (`hebbs-python`).

**Legend:** Supported | Missing | N/A (not applicable)

---

## Operations Overview

| # | Operation | Python SDK Method | One-liner | Status |
|---|-----------|-------------------|-----------|--------|
| 1 | **Remember** | `client.remember()` | Store a new memory with content, importance, context, entity scoping, and causal edges. | Supported |
| 2 | **Get** | `client.get()` | Retrieve a single memory by its ID. | Supported |
| 3 | **Recall** | `client.recall()` | Retrieve memories matching a cue using one or more strategies (similarity, temporal, causal, analogical) with composite scoring. | Supported |
| 4 | **Prime** | `client.prime()` | Pre-load the most relevant memories for an entity at session start (combines temporal + similarity). | Supported |
| 5 | **Revise** | `client.revise()` | Update an existing memory's content, importance, context, or entity — creates a revision snapshot. | Supported |
| 6 | **Forget** | `client.forget()` | GDPR-compliant erasure of memories by entity or specific IDs, with cascade and tombstone tracking. | Supported |
| 7 | **SetPolicy** | `client.set_policy()` | Configure tenant-level policy: snapshot limits, auto-forget threshold, decay half-life. | Supported |
| 8 | **Subscribe** | `client.subscribe()` | Open a real-time stream that pushes relevant memories as conversation context changes. | Supported |
| 9 | **Feed** | `subscription.feed()` | Send text into an active subscription to trigger memory surfacing. | Supported |
| 10 | **CloseSubscription** | `subscription.close()` | Gracefully close an active subscription stream. | Supported |
| 11 | **Reflect** | `client.reflect()` | Trigger the background reflection pipeline — clusters memories and generates insights. | Supported |
| 12 | **GetInsights** | `client.insights()` | Retrieve accumulated insights (insight-kind memories) generated by reflect. | Supported |
| 13 | **Health** | `client.health()` | Check server health, version, memory count, and uptime. | Supported |
| 14 | **Count** | `client.count()` | Return total memory count (convenience wrapper over health). | Supported |
| 15 | **Inspect** | — | Show memory detail + graph neighbors + nearest vector neighbors. | CLI only, not in SDK |
| 16 | **Export** | — | Export memories as JSONL with entity/limit filters. | CLI only, not in SDK |
| 17 | **Metrics** | — | Fetch Prometheus metrics from the HTTP endpoint. | CLI only, not in SDK |
