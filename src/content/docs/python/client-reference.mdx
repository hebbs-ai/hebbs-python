## HebbsClient

The primary interface for interacting with a HEBBS server. All methods are async coroutines.

### Connection

#### `HebbsClient.connect(address, *, timeout=None, tls=None)`

Factory method that returns a connected client instance. Supports `async with` for automatic cleanup.

| Parameter | Type | Description |
|---|---|---|
| `address` | `str` | Server address in `host:port` format |
| `timeout` | `float \| None` | Default timeout in seconds for all operations |
| `tls` | `TlsConfig \| None` | TLS configuration for encrypted connections |

**Returns:** `HebbsClient`

---

### Core Operations

#### `remember(content, entity, *, kind=None, edges=None, metadata=None)`

Store a new memory in the engine.

| Parameter | Type | Description |
|---|---|---|
| `content` | `str` | The memory content to store |
| `entity` | `str` | Entity scope for the memory |
| `kind` | `MemoryKind \| None` | Memory kind (episodic, semantic, procedural) |
| `edges` | `list[Edge] \| None` | Relationships to other memories |
| `metadata` | `dict[str, str] \| None` | Arbitrary key-value metadata |

**Returns:** `Memory`

---

#### `get(memory_id)`

Retrieve a single memory by its ID.

| Parameter | Type | Description |
|---|---|---|
| `memory_id` | `str` | The unique memory identifier |

**Returns:** `Memory`

**Raises:** `HebbsNotFoundError` if the memory does not exist.

---

#### `recall(query, entity, *, strategy="similarity", top_k=10, threshold=None)`

Query the engine for relevant memories.

| Parameter | Type | Description |
|---|---|---|
| `query` | `str` | Natural language query |
| `entity` | `str` | Entity scope to search within |
| `strategy` | `str \| RecallStrategy` | One of `similarity`, `temporal`, `causal`, `analogical` |
| `top_k` | `int` | Maximum number of results |
| `threshold` | `float \| None` | Minimum relevance score (0.0–1.0) |

**Returns:** `RecallOutput`

---

#### `prime(query, entity, *, strategy="similarity", top_k=10)`

Preload relevant memories for low-latency recall.

| Parameter | Type | Description |
|---|---|---|
| `query` | `str` | Natural language query |
| `entity` | `str` | Entity scope |
| `strategy` | `str \| RecallStrategy` | Recall strategy |
| `top_k` | `int` | Maximum number of memories to prime |

**Returns:** `PrimeOutput`

---

#### `revise(memory_id, *, content=None, metadata=None)`

Update an existing memory, creating a new revision linked to the original.

| Parameter | Type | Description |
|---|---|---|
| `memory_id` | `str` | ID of the memory to revise |
| `content` | `str \| None` | Updated content |
| `metadata` | `dict[str, str] \| None` | Updated metadata |

**Returns:** `Memory` (the new revision)

---

#### `forget(entity, *, memory_ids=None)`

Erase memories permanently. Supports targeted or entity-wide deletion.

| Parameter | Type | Description |
|---|---|---|
| `entity` | `str` | Entity scope |
| `memory_ids` | `list[str] \| None` | Specific memories to forget (all if `None`) |

**Returns:** `ForgetResult`

---

### Policy and Reflection

#### `set_policy(entity, *, reflect_enabled=None, reflect_interval=None, reflect_min_memories=None)`

Configure the reflection policy for an entity.

| Parameter | Type | Description |
|---|---|---|
| `entity` | `str` | Entity scope |
| `reflect_enabled` | `bool \| None` | Enable/disable automatic reflection |
| `reflect_interval` | `int \| None` | Minimum seconds between reflections |
| `reflect_min_memories` | `int \| None` | Minimum memory count to trigger reflection |

**Returns:** `None`

---

#### `reflect(entity)`

Trigger an immediate reflection cycle for an entity.

| Parameter | Type | Description |
|---|---|---|
| `entity` | `str` | Entity scope |

**Returns:** `ReflectResult`

---

#### `insights(entity, *, top_k=10)`

Retrieve insights generated by the reflection pipeline.

| Parameter | Type | Description |
|---|---|---|
| `entity` | `str` | Entity scope |
| `top_k` | `int` | Maximum number of insights |

**Returns:** `list[Memory]` (insights are stored as semantic memories)

---

### Streaming

#### `subscribe(entity, *, strategy="similarity", top_k=5, threshold=None)`

Open a real-time subscription that surfaces relevant memories as new context arrives.

| Parameter | Type | Description |
|---|---|---|
| `entity` | `str` | Entity scope |
| `strategy` | `str \| RecallStrategy` | Recall strategy for matching |
| `top_k` | `int` | Maximum memories per push |
| `threshold` | `float \| None` | Minimum relevance score |

**Returns:** `Subscription`

See [Subscribe Streaming](/python/subscribe-streaming/) for full usage.

---

### Health

#### `health()`

Check server health and readiness.

**Returns:** `HealthStatus`

---

#### `count(entity)`

Return the total number of memories stored for an entity.

| Parameter | Type | Description |
|---|---|---|
| `entity` | `str` | Entity scope |

**Returns:** `int`
